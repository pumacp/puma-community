# Code of Conduct

PUMA Community adopts the [Contributor Covenant, version 2.1][cov] as its
code of conduct. All participants in this repository — issues, pull requests,
discussions, and downstream channels — agree to abide by it.

The canonical text is hosted by the Contributor Covenant project at the link
below; we follow it verbatim and do not maintain a separate copy here.

**Canonical text:** <https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

**Reporting concerns:** report any incident privately to
`pumacapstoneproject@gmail.com`. Reports are read only by the active
maintainer team.

The maintainers commit to investigating every report in good faith,
preserving the reporter's confidentiality, and applying the enforcement
ladder defined in the Contributor Covenant.

[cov]: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
